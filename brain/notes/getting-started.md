# Getting Started

1. Put your `.md` notes in the `brain/` folder (subfolders become color groups).
2. Run `bash run.sh` to build the map and open it.
3. Click any star to read that note. Use the search box (top-right) to find anything.
4. Link notes with `[[wiki-links]]` — like this link back to [[welcome]] — and they
   draw a thread between the stars.

For private notes, see the lock in the corner. Back to [[welcome]].
